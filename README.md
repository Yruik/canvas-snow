## 使用方法

### 下载js目录下的canvas-snow.js引入到你的网页即可，无需任何附加操作,如下：
``` html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>下雪了</title>
</head>
<body>
    <h1>hello world</h1>
</body>
<script src="js/canvas-snow.js"></script>
</html>
```
